## Cara Menjalankan Aplikasi

## Instalasi

1. Run semua layanan (Guest Service, Booking Service, Room Service, Payment Service)

2. Instal dependensi:

   ```
   pip install -r requirements.txt
   ```

3. Jalankan frontend:

   ```
   python app.py
   ```

4. Akses aplikasi di browser:
   ```
   http://localhost:5000
   ```
